JCMS Pedigree Tracker

We are very pleased to announce release 1.1.1.3 of our new JCMS Pedigree Tracking tool (JPT). The JPT can be used to connect to your JCMS database and graphically view ancestor and progeny trees. The JPT is a JAVA program that can run on any computer with an ODBC or MySQL connection to your database. The JPT is distributed using JAVA Web Start technology. To get started

Once you have downloaded the application via the Launch button above, check out the Quick Start Guide to help you configure your application and get started.
